## 🔗 Dependencias entre Archivos

```
index.html
    ├─ requiere → styles.css
    ├─ requiere → script.js
    └─ requiere → Font Awesome (CDN)

script.js
    ├─ modifica → index.html (DOM)
    └─ referencia → styles.css (clases)

styles.css
    ├─ estiliza → index.html
    └─ independiente

config.json
    └─ referencia para estructura
```
