## 🔧 Cómo Usar

### 1. **Estructura de Archivos**
```
/
├── index.html          # Archivo HTML principal
├── styles.css          # Estilos CSS
├── script.js           # Funcionalidad JavaScript
└── README.md          # Este archivo
```

### 2. **Instalación**
1. Descargar o clonar los archivos
2. Abrir `index.html` en un navegador web
3. No requiere instalación de dependencias

### 3. **Personalización**

#### Cambiar colores
Editar las variables CSS en `styles.css`:
```css
:root {
    --gold: #d4af37;
    --orange: #f39c12;
    /* ... más colores */
}
```

#### Cambiar número de WhatsApp
Buscar y reemplazar `51960233042` en:
- `index.html` (múltiples ocurrencias)

#### Agregar contenido
1. Duplicar tarjetas existentes en las secciones
2. Actualizar texto e iconos
3. Las animaciones funcionarán automáticamente

#### Agregar imágenes reales
Reemplazar los degradientes SVG/CSS con URLs de imágenes:
```html
<!-- Antes -->
<div class="producto-image" style="background: linear-gradient(...)">

<!-- Después -->
<div class="producto-image" style="background-image: url('tu-imagen.jpg')">
```

### 4. **Agregar Mapa de Ubicación**
En la sección contacto, reemplazar el placeholder con:
```html
<iframe src="https://www.google.com/maps/embed?..." 
        width="100%" height="300" style="border:0; border-radius:8px;" 
        allowfullscreen="" loading="lazy"></iframe>
```
