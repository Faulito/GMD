# ⚡ Guía de Inicio Rápido - GMD GROUP & SOLUTIONS

Bienvenido a tu nuevo sitio web profesional. Esta guía te ayudará a empezar en 5 minutos.

## 📦 Archivos del Proyecto

```
GMD GROUP & SOLUTIONS/
├── index.html              # Página principal (HTML)
├── styles.css              # Estilos y diseño (CSS)
├── script.js               # Interactividad (JavaScript)
├── config.json             # Configuración del sitio
├── README.md               # Documentación completa
├── PERSONALIZACION.md      # Guía de personalización
├── MEJORAS_FUTURAS.md      # Recomendaciones futuras
└── INICIO_RAPIDO.md        # Este archivo
```

## 🚀 Primeros Pasos

### 1. Ver el sitio en tu navegador
1. Abre la carpeta del proyecto
2. Haz doble clic en `index.html`
3. ¡Tu sitio abrirá automáticamente!

### 2. Personalización Básica (2 minutos)

#### Cambiar número de WhatsApp
1. Abre `index.html` con un editor de texto
2. Presiona `Ctrl+H` para buscar y reemplazar
3. Busca: `51960233042`
4. Reemplaza con tu número: `TU_CODIGO_PAIS_Y_NUMERO`
5. Guarda el archivo

#### Cambiar nombre de empresa
1. Abre `index.html`
2. Busca: `GMD GROUP & SOLUTIONS`
3. Reemplaza con tu nombre
4. Guarda

#### Cambiar colores
1. Abre `styles.css`
2. Encuentra la sección `:root` (línea 12)
3. Cambia los valores de `--gold` y `--orange`:
```css
--gold: #d4af37;      /* Color dorado actual */
--orange: #f39c12;    /* Color naranja actual */
```
4. Guarda

### 3. Ver cambios
1. Recarga el navegador (F5)
2. ¡Tus cambios aparecerán!

## 📱 Verificar Responsividad

## 🆘 Problemas Comunes

### El menú hamburguesa no funciona
**Solución:** Asegúrate de que `script.js` está cargado. Presiona F12 y revisa la consola.

### Las imágenes no se ven
**Solución:** Verifica que la ruta sea correcta. Usa rutas relativas: `images/foto.jpg`

### No se ve bien en móvil
**Solución:** Recarga la página completamente (Ctrl+Shift+R)

### El botón WhatsApp no funciona
**Solución:** Verifica que el número esté en formato correcto: `+51960233042` (sin espacios)

## 💡 Tips Útiles

### Para editar código fácilmente
**Recomendado:** Descargar VS Code (gratis)
- https://code.visualstudio.com/
- Abre la carpeta del proyecto
- Edita archivos fácilmente
- Guarda (Ctrl+S)

### Para probar cambios rápido
1. Abre dos ventanas
2. Una con el editor
3. Otra con el navegador
4. Cada vez que guardes (Ctrl+S), recarga el navegador (F5)

### Para capas de contenido
Presiona F12 para abrir herramientas de desarrollador:
- **Inspector:** Click en elemento → editar HTML/CSS en vivo
- **Console:** Ver errores JavaScript
- **Network:** Ver qué recursos se cargan
